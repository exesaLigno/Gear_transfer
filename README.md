# Gear transfer
## Description
(not very) Simple programm designed to transfer files from Linux PC to Samsung Gear S2/Gear S3/Galaxy Watch via Wi-Fi
This programm using open software (nmap, arp, adb) and working only with enabled debug mode on watch.
## Instaling
To compile programm clone repo and go to created folder
```
git clone https://github.com/exesaLigno/Gear_transfer
cd Gear_transfer
```
To compile programm and install it to /usr/bin write
```
make install
```
If you want just compile programm type
```
make
```
